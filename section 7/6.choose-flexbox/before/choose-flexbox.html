When to Use Flexbox instead of CSS Grid

General

  • Better browser support
  • If the items inside of a single parent element needs to flow in just one single direction only.
  • If the children of the single parent element is just content, (content-first). CSS Grid should be reserved primary for adjusting layouts over 2 dimensions (layout-first).
  • You want to give your items complete control over their sizing. CSS Grid by default forces items to fit an individual cell by default.
